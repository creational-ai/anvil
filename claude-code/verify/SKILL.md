# verify

Document verification utility for all Anvil skills.

## Overview

This skill provides document verification capabilities. It checks design and implementation documents for soundness, logical consistency, dependency chain completeness, contradictions, and surprises.

## Quick Reference

| Resource | Path |
|----------|------|
| Guide | `references/verify-doc-guide.md` |

## Commands

- `/verify-doc <path>` - Verify design or implementation document
- `/agent-verify-doc <path>` - Document verification agent (background)

Or use natural language: "Verify this document", "Check this plan for issues"
