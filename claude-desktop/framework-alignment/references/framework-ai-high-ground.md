## Validation Questions

### For Any Project (RAIL)

**Revenue:**
- [ ] Who is paying for this? When?
- [ ] If "piloting," what's the conversion path and timeline?
- [ ] Is there a clear revenue model or are we hoping one emerges?

**Acceleration:**
- [ ] What can we ship to real users in the next 2 weeks?
- [ ] Where is the friction? Can AI remove it?
- [ ] Are we moving at Cursor-speed (daily) or corporate-speed (quarterly)?

**In-Market:**
- [ ] Is this in real users' hands right now?
- [ ] Are we gathering real data from real usage?
- [ ] Or is this a "million-dollar science project" with great demos and zero customers?

**Learning:**
- [ ] What did we learn from users in the last week?
- [ ] Are customers breaking the product? (Good signâ€”we're learning)
- [ ] How fast are we iterating based on feedback?

### For Strategic Positioning

**Layer Check:**
- [ ] What layer are we operating in?
- [ ] If Layer 1-2, do we have massive capital advantage?
- [ ] If Layer 3, are we vertical (own niche) or horizontal (distribution plan)?

**Competitive Reality:**
- [ ] Could a well-funded team copy this in weeks?
- [ ] What's the moat? Learning loops? Relationships? Data?
- [ ] Are we building on top of infrastructure or trying to be infrastructure?

### For Work Composition

**Audit Questions:**
- [ ] What percentage of this project is scripted vs strategic work?
- [ ] What should we automate before the market does it for us?
- [ ] Where is the deep thinking time? The daydreaming?

### For Three Rs

**Rigor:**
- [ ] Do we have deep domain mastery or surface knowledge?
- [ ] Have we evaluated the edge cases that break other solutions?
- [ ] Would experts in this domain trust our work?

**Relationships:**
- [ ] Are we generating trust or just generating output?
- [ ] What relationships are we building through this work?
- [ ] Who will advocate for us based on how we've treated them?

**Resilience:**
- [ ] What's our pivot plan if current approach fails?
- [ ] Are we prepared to stay in the game after setbacks?
- [ ] Is our identity tied to this specific wave, or are we the ocean?
